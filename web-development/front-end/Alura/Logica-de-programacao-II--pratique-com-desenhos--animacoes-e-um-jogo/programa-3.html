Lista de comandos:

  • Clique com o botão esquerdo do mouse para escolher a cor;
  • Clique com o botão direito do mouse para desenhar um círculo;
  • Clique com o botão esquerdo do mouse, segurando a tecla shift do teclado, para aumentar o tamanho do círculo;
  • Clique com o botão esquerdo do mouse, segunrando a tecla ctrl do teclado, para diminuir o tamanho do círculo;
  • Clique e segure para desenhar continuamente. Também são válidas aqui as opções de aumentar e diminuir o tamanho do círculo;
